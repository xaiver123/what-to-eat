# 今天吃什么 - 产品需求文档

## 项目概述
- 项目名称：今天吃什么
- 项目类型：Web 应用
- 核心特色：AI 小精灵对话助手

## 功能需求

### 1. AI 小精灵
- 页面居中显示小精灵图片（预留 sprite.png 占位符）
- CSS 上下浮动呼吸动画
- 对话气泡框显示 AI 回复

### 2. 食物库功能
- 用户可以添加食物到列表
- 数据存储在浏览器 LocalStorage 中
- 显示已添加的食物列表

### 3. 随机抽取功能
- "帮我抽一个！"按钮（默认隐藏，开场白后显示）
- 本地随机抽取食物
- AI 对抽取结果进行评价

### 4. DeepSeek API 集成
- 后端调用 DeepSeek API（Base URL: https://api.deepseek.com，模型：deepseek-chat）
- 系统提示词："你是一个贪吃的魔法小精灵，语气可爱活泼"
- 评价提示词："用户今天决定吃【抽中的食物】，请你以可爱小精灵的语气，对这个食物做一句有趣的评价或描述，字数不超过50字"

## 技术栈
- 前端：纯 HTML + CSS + 原生 JavaScript
- 后端：Node.js + Express + dotenv
- API 调用：前端 fetch → 后端 → DeepSeek API

## 交互流程
1. 用户打开网页 → 前端调用后端 → AI 生成欢迎语 → 显示气泡框和按钮
2. 用户点击按钮 → 本地随机抽取食物 → 气泡框显示"正在施展美食魔法..."
3. 前端发送食物名到后端 → AI 生成评价 → 更新气泡框显示结果
